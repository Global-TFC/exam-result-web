# Google Sheets storage options

## Option A: One Spreadsheet per Class (recommended)

### Pros
- Easy teacher ownership and permission control.
- Lower risk of editing wrong class data.
- Clear separation of responsibilities.

### Cons
- More links/config entries to manage.

## Option B: One Spreadsheet with multiple tabs

### Pros
- Only one spreadsheet ID to manage.
- Centralized structure.

### Cons
- Higher chance of accidental edits in other class tabs.
- Permissions are harder per class (spreadsheet-level sharing).

## Final recommendation

For madrasa/school workflow with different class teachers, **Option A** is usually safer and easier.
